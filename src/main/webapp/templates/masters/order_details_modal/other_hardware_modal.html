Want to add more Hardware from Inventory?

To add more Hardware from Inventory click on buttons present below for following functions :

  • To Add Complete New Hardware from Inventory, Click on "Add New Hardware from Inventory".
  • To Add Hardware from Inventory Fronts By Pre Fetching Values from previously Entered Hardware from Inventory, Click on "Add Hardware from Inventory with Value".
  • To Get Back To Order Details Page, Click on "Done".
